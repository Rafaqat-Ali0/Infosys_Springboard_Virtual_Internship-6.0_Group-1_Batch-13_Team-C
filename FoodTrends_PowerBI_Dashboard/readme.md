# Power BI Dashboards

---

## Overview
This folder contains the dashboard outputs developed as part of the FoodTrends project.

The dashboards collectively present insights into customer preferences, dietary patterns, nutrient analysis, and lifestyle impact within the food and beverage domain.

---

## Contents
- Dashboard visuals exported from Power BI  
- Each file represents a specific analytical view of the dataset  

---

## Purpose
The objective of these dashboards is to provide a clear and visual understanding of data-driven insights, enabling better interpretation of trends and patterns.

---

## Note
All dashboards are created using Power BI and are part of the overall project analysis.
